# Cairo Basics Repository

Welcome to the **Cairo Basics** repository! This repository contains a collection of resources, examples, and foundational concepts for learning Cairo, a Turing-complete programming language for writing provable programs, particularly on the StarkNet platform.

## What's Included

- **Introduction to Cairo**: An overview of Cairo, its purpose, and use cases in blockchain programming.
- **Basic Syntax**: Learn the fundamentals of Cairo's syntax, including data types (`felt252`, `struct`, `enum`, etc.), functions, and control flow.
- **Working with Arrays**: How to declare, initialize, and manipulate arrays in Cairo, including iteration and element access.
- **Enums and Matching**: A detailed guide on working with `enum` types, pattern matching, and variant extraction.
- **Structs and Tuples**: Explanation and examples of creating and using structs and tuples in Cairo programs.
- **Writing Functions**: Best practices for defining and calling functions, passing parameters, and returning values.
- **Conditionals and Loops**: Examples of `if`, `else`, `while`, and `for` loops, and how they work in Cairo.
- **Memory Management**: Basics of memory allocation and pointer management in Cairo.
- **Debugging and Testing**: How to test and debug Cairo programs effectively, including using `starknet::println` or similar for logging.
